# India Cards — Claude Code Plugin Marketplace

A Claude Code plugin marketplace containing the **india-cards** plugin.

## Install

Add this marketplace in the Claude app, then install the plugin:

```
/plugin install india-cards@aswinsamba/india-cards-plugin
```

## Plugins

| Plugin | Description |
|---|---|
| `india-cards` | Analyse Indian bank statements and recommend the best credit cards based on your spend patterns |
